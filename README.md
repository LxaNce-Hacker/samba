### Features

- Works Wired / Wireless.
- Support Termux / Linux 
- Beginners friendly


### Installation

- Just, Clone this repository -
```
$ git clone https://github.com/LxaNce-Hacker/samba
```

- Change to cloned directory and run `connect.sh` -
```
$ cd samba
$ bash connect.sh
```
